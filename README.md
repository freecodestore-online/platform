# FreeCodeStore

**[freecodestore.pages.dev](https://freecodestore.pages.dev)** — Free developer tools. Regex tester, JSON formatter, diff checker, API tester, and 20 more. No signup, runs in your browser.

## Tools

### Formatters
- JSON Formatter & Validator (tree view, diff, convert to YAML/TS types)
- SQL Formatter & Playground
- Markdown Editor & Preview (GFM, syntax highlighting)
- CSS/JS/HTML Minifier & Beautifier
- .env File Editor

### Encoders/Decoders
- Base64/URL/HTML/Hex/Binary encoder
- JWT Token Decoder (verify, generate)

### Generators
- UUID/ULID/nanoid Generator (6 formats, bulk)
- Hash Generator (MD5, SHA-1/256/384/512, HMAC)
- Lorem Ipsum & Mock Data Generator
- CSS Grid Generator

### Testers
- Regex Tester & Debugger (live highlighting, explain mode)
- Diff Checker (side-by-side, inline, syntax highlight)
- Cron Expression Builder & Parser
- API Request Tester (like mini-Postman)
- HTTP Status Code Reference

### Converters
- Color Format Converter (hex/RGB/HSL/CMYK)
- Unix Timestamp Converter
- YAML ↔ JSON
- URL Parser & Builder
